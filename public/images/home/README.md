# Homepage Image Slots

Replace these files when real Hell Cemetery screenshots or key art are ready. Keep the same filenames to update the homepage without changing React code.

- `hero-cemetery-world.png`: main homepage hero background.
- `metroidvania-exploration.png`: exploration/platforming card art.
- `action-platformer-combat.png`: combat card art.
- `world-lore-archive.png`: lore/world-history card art.

Recommended replacement format:

- Landscape 16:9 images.
- At least 1600px wide for the hero.
- Keep important subjects away from the center hero title area.
- Avoid text inside images; the website renders text separately for SEO and translation.
